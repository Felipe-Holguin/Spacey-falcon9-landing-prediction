# SpaceY – Falcon 9 First Stage Landing Prediction

## Project Overview
The objective of this project is to predict whether the Falcon 9 first stage will land successfully, allowing estimation of launch costs.

This project is a modified and extended version of the IBM Data Science Capstone project from Coursera. The main goal is to build my first end-to-end data science project, applying concepts while adapting and improving the original workflow.
